A Buildscreen for TeamCity servers

RedGreen aims to provide a clean visual interface for statuses of builds on TeamCity servers along with build meta information. This is a work in progress, we are beta-ish stable 👩‍💻.

Features

  • Free 🥳 (MIT Licence)
  • List latest completed builds with meta infomation.
  • Fits narrow viewports and can sneak into busy info screens.
  • Filter which builds to display.
  • Error tracking using Sentry.io with their standard sensitive data scrubbing 😇
  • Supports Team City 2019.1 or newer.

Are you looking for another feature? Feel free to make an issue on GitHub. No promises though, this is a side-project. We encourage contributions, but let's discuss the changes upfront.

Notes on security

We use TeamCity's token-based authentication.

  • Set up a read only user in TeamCity for this buildscreen.
  • Generate token for that user and use to sign into build screen. Token will be stored in browser localstorage.
  • If your build data contains or is considered sensitive information, don't have a build screen.

Can't log in or get data? (CORS)

Your TeamCity server must accept requests from where ever you host the buildscreen. Please check TeamCity docs on how to enable CORS.
