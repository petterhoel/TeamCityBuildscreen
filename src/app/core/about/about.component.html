See Your TeamCity Builds

RedGreen aims to provide a clear visualization of build statuses from your TeamCity installation. This is a work in progress, we are beta-ish stable. Should you find a 🐛, please file an issue on GitHub 👩‍💻

Features

  • Lists the latest completed builds on default branch with status symbol, author, time stamp, commit hash and commit message.
  • Fits narrow viewports and can sneak into busy info screens.
  • Filter which builds to display.
  • Supports Team City 2019.1 or newer.
  • Free – MIT Licence 🥳

Are you looking for another feature? Feel free to make an issue on GitHub. No promises though, this is a side-project. We do encourage contributions. In any case, let's discuss the changes upfront, before anyone commits any time, and align our expectations 👍.

On Privacy 🧐

RedGreen uses Sentry.io for automatic error tracking, with their standard sensitive data scrubbing 😇

On Security 🔐

We use TeamCity's token-based authentication. This is a variety of basic authentication. Therefore we need to store your token on the client and send it along with every request. For your security we suggest the following:

  • Set up a read only user in TeamCity for this build screen.
  • Generate a token for that user and use it to sign in. This token will be stored in your browser's localstorage and is readable and therefore steal-able by javascript. But since it is a read only user your exposure is limited 😇
  • However, if your build data contains or is considered sensitive information, don't have a build screen 🙀

Can't log in or get data? (CORS) 🤔

Your TeamCity server must accept requests from https://redgreen.app or where ever you host your copy of this buildscreen. Please check the TeamCity docs on how to enable CORS.
